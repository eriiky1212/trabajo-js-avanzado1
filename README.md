# TRABAJO FINAL – JavaScript Avanzado

## Descripción
Este proyecto corresponde al Trabajo Final del módulo de JavaScript Avanzado.

Consiste en el desarrollo de un sitio web completo para una empresa ficticia: NovaWeb Studio.

El objetivo del trabajo es aplicar HTML5, CSS3, JavaScript y AJAX, incorporando funcionalidades dinámicas e interactividad en distintas secciones del sitio web.

---

## Tecnologías utilizadas
HTML5  
CSS3  
JavaScript (ES6)  
AJAX – Fetch API  
Leaflet.js  
OpenStreetMap  

---

## Páginas del sitio

Inicio (index.html)  
- Página principal del sitio web  
- Estructura semántica  
- Mínimo cuatro secciones  
- Noticias cargadas dinámicamente desde archivo JSON mediante AJAX  

Galería (galeria.html)  
- Galería dinámica desarrollada con JavaScript  
- Navegación interactiva  

Presupuesto (presupuesto.html)  
Formulario dividido en dos partes:

Datos de contacto  
- Validación mediante JavaScript  
- Restricciones de formato y longitud  

Cálculo de presupuesto  
- Selección de producto  
- Plazo configurable  
- Extras opcionales  
- Actualización automática del total sin recargar la página  

Contacto (contacto.html)  
- Integración de mapa dinámico  
- Leaflet + OpenStreetMap  
- Visualización de ubicación

## Ejecución del proyecto
Este proyecto utiliza JavaScript y carga dinámica de datos mediante AJAX (Fetch API).

Para su correcta visualización:

1. Descomprimir el archivo ZIP  
2. Abrir la carpeta del proyecto en Visual Studio Code  
3. Ejecutar utilizando la extensión Live Server  

Nota:
Si el proyecto se abre directamente mediante doble clic (file://), algunas funciones dinámicas pueden no ejecutarse correctamente.
